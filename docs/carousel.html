name values default description
data-type normal | loop | infinite normal normal : se bloque au début et à la fin
loop : revient au début ou à la fin
infinite : ne s'arrête jamais
data-pagination Active la pagination.
data-darg Active le drag n drop.
data-auto number (ms) Active et défini l'interval entre chaque animation.
