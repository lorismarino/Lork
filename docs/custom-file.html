CustomFile

exemple

code

 <div class="custom-file" data-type="files" data-name="test-custom-file" data-label="Choose a file."
data-button="Choose" data-size="true">
</div>
name values default description
data-name string Utilisé pour le name de l'input.
data-label string Utilisé comme titre informatif.
data-button string Utilisé comme texte sur le bouton.
data-size Affiche la taille du fichier.
data-type file | files | folders file file : ajout d'un fichier
files : ajout de plusieurs fichiers
folder : ajout de dossier
