Forms

exemple

code

<form class="validation" novalidate>
  <div class="input-group" data-minlength="4" data-errorminlength="minimum 4 caractères" data-required="true"
    data-requiredmessage="champ obligatoire">
    <label for="name">Name</label>
    <input type="text" id="name">
  </div>
</form>

form

Vous pouvez ajouter la classe .validation sur le form pour avoir une validation des champs.

Vous pouvez ajouter l'attribut data-nosubmit pour ne pas avoir la validation à la soumission ou l'attribut data-noinstant pour ne pas avoir la validation à la volée.

input-group

name description
data-required Rend le champ obligatoire.
data-minlength Demande un minimum de charactères.
data-errorminlength Message d'erreur si min length non respecté.
data-maxlength Demande un maximum de charactères.
data-errormaxlength Message d'erreur si max length non respecté.
data-errormessage Message d'erreur si le format du champ n'est pas respecté.
