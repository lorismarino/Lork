> 768 720px -s

container

.container utlisera la max-width en rapport avec la résolution de l'écran.

.container-fluid prendra toute la largeur.

columns

pour activer la grille, utilisez la class .row

pour chacune de vos colonnes, utilisez .col{prefix}{col}

pour des colonnes de taille égal, utilisez .col-equal{prefix}

pour des colonnes de taille auto, utilisez .col-auto{prefix}

alignement

ajoutez ces classe sur la classe .row pour avoir l'alignement

.justify-content{prefix}{start|center|end|around|between|evenly} : ajoute le centrage horyzontal de vos colonnes

.align-items{prefix}{start|center|end|stretch} : ajoute le centrage vertical de vos colonnes
